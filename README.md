**Client:** Javascript, Jquery, Twig, Sass, Materialize CSS

**Server:** PHP, Symfony Framework

**API's & external services:** OpenCage (Geocoding API), Stripe API, Google OAuth 2.0 API, Mailtrap, SurveyMonkey
